# spotBot #
## Telegram Bot that Download songs from Spotify using spotDL Binary ##

### Bot in action ###
![demo](.github/assets/demo.png)


### Features ###

- Download FLAC / MP3 songs / playlist / album from Spotify.

### Setup ###

`
bash setup.sh
`

### Use Bot ###

- Fill in bot token inside bot.py and then:

`
python bot.py
`

and see the magic.
